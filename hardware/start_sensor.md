# 🚦 출발선 센서 + 메인 허브 — 라디오 그룹 2

## 역할
- PC와 USB 연결되어 시리얼로 STANDBY / START / FINISH 전송
- 초음파 센서로 차량 감지 (디바운싱 적용)
- 도착선에서 라디오로 FINISH 수신 → 시리얼 전달

## MakeCode 의사코드

```
변수:
  현재상태 = "IDLE"
  감지시작시간 = 0
  감지중 = false

시작할 때:
  라디오 그룹 설정(2)
  시리얼 리다이렉트(USB)

무한반복:
  거리 = 초음파거리읽기(P1=Trig, P2=Echo)  // cm 단위
  
  // ========== 디바운싱 로직 ==========
  // 거리 < 15cm가 0.2초 이상 지속될 때만 유효 감지
  
  IF 거리 < 15:
    IF 감지중 == false:
      감지시작시간 = 경과시간(밀리초)
      감지중 = true
    ELSE:
      경과 = 경과시간(밀리초) - 감지시작시간
      
      IF 경과 >= 200:  // 0.2초 이상 지속 = 유효 감지
        // ========== 상태 전환 로직 ==========
        IF 거리 < 10 AND 현재상태 != "STANDBY":
          현재상태 = "STANDBY"
          시리얼 출력("STANDBY")
        
  ELSE:  // 거리 >= 15cm
    감지중 = false
    
    IF 현재상태 == "STANDBY":
      현재상태 = "RACING"
      시리얼 출력("START")
      일시정지(3000ms)  // 3초 쿨타임 (재감지 방지)

// ========== 라디오 수신 ==========
라디오 수신할 때(텍스트):
  IF 텍스트 == "FINISH":
    시리얼 출력("FINISH")
    현재상태 = "IDLE"
```

## 핀 연결
| 핀 | 연결 |
|---|---|
| P1 | 초음파 센서 Trig |
| P2 | 초음파 센서 Echo |
| USB | PC 시리얼 연결 |

## 디바운싱 설명
- **목적**: 초음파 센서의 노이즈/가비지 데이터 필터링
- **방식**: 거리 < 15cm 상태가 **연속 0.2초(200ms)** 이상 유지될 때만 유효한 감지로 인정
- **효과**: 순간적인 물체 감지(손 지나감 등)를 무시하고 차량이 실제로 선에 있을 때만 반응
