# 🏎️ 링비트 카 (수신) — 라디오 그룹 1

## 역할
조이스틱에서 수신된 X, Y 값에 따라 좌/우 모터 속도를 매핑합니다.

## MakeCode 의사코드

```
시작할 때:
  라디오 그룹 설정(1)

라디오 수신할 때(텍스트):
  값 배열 = 텍스트.분리(",")
  X = 숫자로 변환(값배열[0])  // 0~1023
  Y = 숫자로 변환(값배열[1])  // 0~1023
  
  // Y를 전진/후진 속도로 매핑 (-100 ~ +100)
  속도 = 매핑(Y, 0, 1023, 100, -100)
  
  // X를 조향값으로 매핑 (-50 ~ +50)
  조향 = 매핑(X, 0, 1023, -50, 50)
  
  // 좌/우 모터 속도 계산
  왼쪽모터 = 속도 + 조향
  오른쪽모터 = 속도 - 조향
  
  // 범위 제한 (-100 ~ 100)
  왼쪽모터 = 범위제한(왼쪽모터, -100, 100)
  오른쪽모터 = 범위제한(오른쪽모터, -100, 100)
  
  // 모터 제어 (Ring:bit Car 확장 블록)
  ringbitcar.모터제어(왼쪽모터, 오른쪽모터)
```

## 핀 연결
| 핀 | 연결 |
|---|---|
| P1 | 왼쪽 모터 (Ring:bit 기본) |
| P2 | 오른쪽 모터 (Ring:bit 기본) |

## 참고
- Ring:bit Car V2 확장 블록 필요
- 데드존 추가 권장: |속도| < 10이면 정지
