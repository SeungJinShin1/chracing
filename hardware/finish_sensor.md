# 🏁 도착선 센서 — 라디오 그룹 2

## 역할
- 단독 전원 (배터리)
- 초음파 센서로 차량 통과 감지 → 라디오로 FINISH 전송
- 디바운싱 + 3초 쿨타임으로 중복 감지 방지

## MakeCode 의사코드

```
변수:
  감지시작시간 = 0
  감지중 = false

시작할 때:
  라디오 그룹 설정(2)

무한반복:
  거리 = 초음파거리읽기(P1=Trig, P2=Echo)
  
  // ========== 디바운싱 로직 ==========
  IF 거리 < 20:
    IF 감지중 == false:
      감지시작시간 = 경과시간(밀리초)
      감지중 = true
    ELSE:
      경과 = 경과시간(밀리초) - 감지시작시간
      
      IF 경과 >= 200:  // 0.2초 이상 지속 = 유효 감지
        라디오 전송("FINISH")
        LED 아이콘(체크마크)
        일시정지(3000ms)  // 3초 쿨타임 (중복 감지 방지)
        감지중 = false
        LED 초기화()
  ELSE:
    감지중 = false
```

## 핀 연결
| 핀 | 연결 |
|---|---|
| P1 | 초음파 센서 Trig |
| P2 | 초음파 센서 Echo |

## 주의사항
- 배터리 전원으로 독립 운용
- 3초 쿨타임으로 같은 차가 도착선을 여러 번 트리거하는 것을 방지
- 디바운싱 0.2초로 센서 노이즈 필터링
