# 🕹️ 조이스틱 비트 (송신) — 라디오 그룹 1

## 역할
조이스틱의 X, Y 아날로그 값을 읽어 라디오 채널 1로 상시 전송합니다.

## MakeCode 의사코드

```
시작할 때:
  라디오 그룹 설정(1)

무한반복:
  X값 = 아날로그 읽기(P1)  // 0 ~ 1023
  Y값 = 아날로그 읽기(P2)  // 0 ~ 1023
  
  // "X,Y" 형식으로 전송
  텍스트 = X값 + "," + Y값
  라디오 전송(텍스트)
  
  일시정지(50ms)  // 20Hz 전송 주기
```

## 핀 연결
| 핀 | 연결 |
|---|---|
| P1 | 조이스틱 X축 |
| P2 | 조이스틱 Y축 |

## 참고
- 센터값은 약 512 (조이스틱 중립)
- 라디오 그룹 1은 주행 전용 (센서와 분리)
